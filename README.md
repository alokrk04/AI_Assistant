🤖 Flask Chatbot with Google Gemini API
A lightweight, interactive web-based chatbot application built with Python (Flask) and Google's Gemini 2.5 Flash model.
It features a responsive Single Page Application (SPA) interface using vanilla JavaScript, HTML, and CSS.

🚀 Features
Real-time Chat Interface: Clean, messenger-style UI.

AI Integration: Powered by Google's Gemini API for intelligent responses.

Responsive Design: Works on desktop and mobile browsers.

Lightweight Backend: Built on Flask.

Dynamic Styling: Distinct styles for user and bot messages.

🛠️ Tech Stack
Backend: Python 3, Flask

AI Model: Google Generative AI (Gemini 2.5 Flash)

Frontend: HTML5, CSS3, JavaScript (Fetch API)
